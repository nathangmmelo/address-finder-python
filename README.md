# 🗺️ Buscador de CEP

Aplicação de linha de comando em Python para consulta de endereços via API ViaCEP.

## 🚀 Funcionalidades

- Buscar endereço completo a partir de um CEP
- Buscar CEPs pelo nome da rua, cidade e estado
- Tratamento de erros de conexão e entradas inválidas

## 🛠️ Tecnologias

- Python 3
- Biblioteca `requests`
- API pública [ViaCEP](https://viacep.com.br)

## ▶️ Como usar

1. Clone o repositório:
```
git clone https://github.com/nathangmmelo/address-finder-python.git
```

2. Instale a dependência:
```
pip install requests
```

3. Execute:
```
python "Cep e endereço.py"
```

- Consumo de APIs REST com Python
- Manipulação de dados JSON
- Tratamento de exceções
